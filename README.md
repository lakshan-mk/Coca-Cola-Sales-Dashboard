# Coca-Cola Sales Dashboard (Excel)

## Overview
This project is an interactive sales dashboard built using Microsoft Excel to analyze Coca-Cola sales data. 
It provides insights into revenue performance across regions, products, and time periods.

## Tools & Technologies
a) Microsoft Excel
b) Pivot Tables & Pivot Charts
c) VBA (Workbook Automation)
d) Data Cleaning & Transformation

## Key Features
a) Automated dashboard navigation using VBA
b) Sales analysis by region and product
c) Interactive visualizations for decision-making
d) Clean separation of raw data, analysis, and dashboard layers

## Business Insights
a) Identified top-performing regions and products
b) Analyzed sales contribution patterns
c) Enabled management-level reporting using Excel dashboards

## Use Case
Designed as a business analytics solution for sales teams and decision-makers.
